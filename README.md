# ai_cv

Technical learning roadmap for **AI and Computer Vision**, based on the **University of Oviedo (Uniovi), Spain** microcredential:

**Desarrollo de Aplicaciones con Inteligencia Artificial para Visión por Computador**

This repository is organized as a single structured timeline, from Python foundations to a final computer vision API.

## Purpose

This repository is used to study and organize the main topics covered by the microcredential:

- Python refresher
- computer vision foundations
- neural network basics
- object detection training and fine-tuning
- practical vision applications
- vision-language integration
- deployment and integration
- final capstone API
